# react-ssr-demo

このプロジェクトは、React SSRのボイラープレートです。

主な用途は、AIを用いたコーディングの練習、およびAIを活用して実用的な小規模ツールを実装することで、プロンプトエンジニアリングの能力を向上させることです。

## 技術スタック

- React 19
- Webpack 5

## 実行環境

- Node.js v22+

## 手動で実装したモジュール

- ボイラープレートのコアコード
- コーディング規約
- モジュール分割、ファイル構造
- コントローラー層

## AIによって実装したモジュール

- コンポーネント
- Hooks
- 共通メソッド
- モデル層

## 使用方法

1. `npm i` でインストール
2. `npm run dev` を実行し、`http://127.0.0.1:9090` を開く
3. `npm run build` でビルド

## 外部依存関係

1. [ollama](https://ollama.com/download)
